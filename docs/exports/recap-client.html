À recevoir →

Prochaine étape : votre validation visuelle. Une fois le visuel acté, on remplace votre site Hostinger par la vitrine en vrai code (Laravel), puis on déploie l'outil de suivi des passages, l'écran le plus important : il marche hors ligne et photographie chaque intervention.
