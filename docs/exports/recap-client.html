Ce que la plateforme contient

Trois espaces, un seul outil.

Une vitrine publique pour attirer, votre espace métier privé pour suivre, un portail client pour rassurer. Une seule connexion oriente chacun vers le bon endroit : mot de passe pour vous, lien magique pour vos clients.

1

Vitrine — pour vos visiteurs

Ce que voient prospects, conciergeries et villas de location. Public, indexé par Google, WhatsApp d'un clic.

2

Votre espace — pour vous

Saisie sur le terrain même hors ligne, pilotage au bureau. Privé, protégé par votre mot de passe.

3

Espace client — pour vos clients

Le dernier passage, les mesures, les photos. En lecture seule, accessible par un lien magique sans mot de passe.
