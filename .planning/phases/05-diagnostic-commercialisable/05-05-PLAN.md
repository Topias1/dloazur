<objective>
Build brick 2's delivery half — the downloadable PDF report (S8) — plus close the two route-level Nyquist gaps (Req8 PDF + Req9 public route). The PDF is generated synchronously on download via spatie/laravel-pdf's DomPDF driver (no queue, no Node/Chrome — fits scale-to-zero Laravel Cloud, D-05), rendering the full diagnostic (pool info, measurements/problem, ordered action plan with doses/products, the ambre safety block, the confidence, and the mandatory disclaimer + footer). That keeps it fully parallel with Plan 04 in Wave 3 (zero shared files). The PDF route name (diagnostic.pdf) is created here and consumed by Plan 04's link.

Purpose: deliver Req8 (PDF) + Req9 (public route) + D-06 (session-gated anonymous PDF access, mitigating enumeration).
