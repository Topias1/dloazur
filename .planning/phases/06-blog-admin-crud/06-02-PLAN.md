|----------|-------------|
| Googlebot/visitor → /blog public read | Untrusted slug crosses into find()/Post query |
| cache store → app | serializable_classes=false: any cached object becomes __PHP_Incomplete_Class |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-06-04 | Denial of Service | cached payload deserialize crash | mitigate | cacheablePayloadFromDb returns plain scalar arrays, Carbon→ISO-8601; regression test asserts serialize round-trip survives allowed_classes=false (commit 50f9929 class) |
| T-06-05 | Information Disclosure | draft posts leaking to sitemap/public | mitigate | scopePublished() applied in DB read path; test asserts sitemap + all() exclude drafts (Pitfall 6) |
| T-06-06 | Tampering | slug route injection | mitigate | public blog.show route already constrains slug to `[a-z0-9-]+`; Post query is parameterized (Eloquent) |
| T-06-SC | Tampering | composer/npm installs | accept | this plan adds NO new package |
</threat_model>
