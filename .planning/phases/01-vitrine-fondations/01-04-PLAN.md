submit() flow:
1. $this->rateLimit(5, 60) — danharrin trait throws TooManyRequestsException if exceeded
2. catch TooManyRequestsException → addError('throttle', 'Trop d\\'essais. Attendez quelques minutes puis réessayez.'); return
3. $this->protectAgainstSpam() — Spatie trait throws SpamException if honeypot tripped (silent rejection from user POV — set $sent=true to fake success)

5. Mail::to(config('contact.recipient'))->send(new ContactMessage(name, email, phone, message))
6. $this->sent = true; $this->reset(['name', 'email', 'phone', 'message'])
