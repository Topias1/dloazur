# Phase 999.1 Plan 03: Service-Page Architecture + BreadcrumbSchema Summary

One-liner: BreadcrumbSchema builder + layout wiring + 3 new service Blade pages (entretien, analyse-eau, spa) + eau-verte expanded to 1600 words with Alpine FAQ accordion.

## What Was Built

### Task 1 — BreadcrumbSchema + layout wiring (`74c4ca9`)

- Created `BreadcrumbSchema.php` (`final class`, `toScript(array $crumbs): string`) in the same `App\Support\SchemaOrg` namespace as `LocalBusinessSchema`. Uses `Schema::breadcrumbList()->itemListElement([...items...])->toScript()`.
- Added `:type="$type ?? 'website'"` to `<x-seo.meta>` call in `layouts/app.blade.php` — forwards `$type` so blog pages emit `og:type="article"`.
- Added `$breadcrumbJsonLd` emission slot in `layouts/app.blade.php` immediately after the existing `$jsonLd` slot. This is the **single** place `BreadcrumbList` is emitted across the whole app — pages must never `@push` it.
- 6 BreadcrumbTest assertions pass: builder output correctness + layout wiring.

### Task 2 — Controller + routes (`e16d03e`)

- Injected `BreadcrumbSchema` into `eauVerteUrgence()` (existing method now passes `breadcrumbJsonLd`).
- Added three new controller methods with per-method DI: `entretienRecurrent()`, `analyseEau()`, `spa()`. Each passes `title`, `description`, `canonical`, `ogImage`, and `breadcrumbJsonLd` with a 3-crumb trail (Accueil › Services › [page]).
- Registered `services.entretien-recurrent`, `services.analyse-eau`, `services.spa` inside the existing `cache.headers:vitrine` group in `routes/vitrine.php`.
- Stub Blade views created so routes immediately resolve (Task 3 fleshes them out).
- `VitrinePagesTest` added: 200 checks, canonical presence, BreadcrumbList presence, h1 count, duplicate-emission regression guard (`substr_count($content, '"@type":"BreadcrumbList"') === 1`).

### Task 3 — Service page content + eau-verte expansion (`6fbe47b`)

**entretien-recurrent.blade.php**: Navy-900 hero band with kicker, h1, subtitle, CTA row (azure + WhatsApp), visible breadcrumb nav (`aria-label="Fil d'Ariane"`). Content: 3 checklist sections (nettoyage, chimie, équipements), formules section, local expertise, `<!-- [À compléter — Pierre ADAM] -->` for tarifs.

**analyse-eau.blade.php**: Navy-900 hero. Content: tropical climate rationale (quoting Zyro article stats: pH efficiency -50%), 4-parameter grid cards, professional method section, 6 real-world use cases, `<!-- [À compléter — Pierre ADAM] -->` for kit/brand details.

**spa.blade.php**: Navy-900 hero, spa/jacuzzi-specific content (D-11). Content: tropical spa specifics, 3 protocol sections (traitement, filtration, équipements), maintenance frequency advice, installation section, `<!-- [À compléter — Pierre ADAM] -->` for spa brands/types.

**eau-verte-urgence.blade.php** (expanded from 124 to 337 lines, 1636 words): Added navy-900 hero band with visible breadcrumb, "Causes de l'eau verte" section (5 causes), "Checklist DIY" with 4 DIY steps, "Protocole 5 étapes" with colored step circles (navy/azure/lagon), before/after parameter table (pH/chlore/TAC/aspect), Alpine FAQ accordion (3 questions with `x-data`, `x-collapse`, `<button>` triggers, chevron rotation, `:aria-expanded`).
