## Deviations from Plan

**1. [Rule 1 - Bug] blog dates were already non-placeholder**
- Found during: Task 2 — dates were 2025-02-10 and 2025-03-15, not 2025-01-01
- Issue: The plan spec said "Replace placeholder `date: 2025-01-01`" but earlier plan work had already moved them to Zyro metadata dates
- Fix: No file change needed; the test correctly verifies no `2025-01-01` remains; documented as Pierre-action (dates should be verified against real first-publish date)
- Files modified: none (pre-existing state was already correct)

**2. [Rule 2 - Missing] `show_date` guard removed from Article JSON-LD datePublished**
- Found during: Task 2 — prior code only emitted datePublished when `show_date: true`
- Issue: `show_date` controls *display* in the UI, not structured data validity; schema.org structured data should have datePublished when a real date is available
- Fix: Removed the `if ($post['show_date'])` gate in `buildArticleSchema()`, always emit datePublished + dateModified
- Files modified: app/Http/Controllers/BlogController.php
